# addon 프롬프트: 웹 서비스

API 서버, 대시보드, annotation UI, admin tool, 내부 웹 도구에 붙이는 추가 기준이다.

## 프롬프트

```text
웹 서비스 기준을 추가로 적용해라.

- 사용자 입력은 신뢰하지 말고 백엔드 검증을 요구해라.
- 인증, 인가, 입력값 검증, API 계약을 확인해라.
- 요청/응답 스키마와 실패 응답 형식을 명확히 하게 해라.
- 내부 예외 상세, 토큰, 개인정보, 내부 URL이 로그나 응답에 노출되지 않게 해라.
- 환경변수와 secret 누락 시 안전하게 실패하는지 확인해라.
- 프론트엔드는 로딩, 실패, 빈 상태를 처리하게 해라.
- 권한 체크를 화면 표시 여부에만 의존하지 않게 해라.
- SQL injection, XSS, CSRF, 파일 업로드 위험을 검토해라.
```
