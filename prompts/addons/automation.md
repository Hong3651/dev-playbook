# addon 프롬프트: 자동화 스크립트

파일 처리, 로그 정리, 데이터 변환, 실험 보조, 반복 작업 자동화에 붙이는 추가 기준이다.

## 프롬프트

```text
자동화 스크립트 기준을 추가로 적용해라.

- 입력 파일, 입력 폴더, 실행 인자를 명확히 해라.
- 출력 위치와 덮어쓰기 정책을 확인해라.
- 실패 시 원인을 알 수 있는 메시지를 요구해라.
- 경로와 설정값은 상수, 실행 인자, 설정 파일 중 하나로 분리해라.
- 단일 파일로 충분하면 과한 구조를 만들지 마라.
- 외부 API, DB, 장비 연결이 생기면 automation만으로 충분한지 다시 판단해라.
- 샘플 데이터와 로그에 민감정보가 없는지 확인해라.
- 실행 명령과 최소 검증 예시를 남기게 해라.
```
